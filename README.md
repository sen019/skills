# AntStack Skills

A curated collection of AI agent skills for use with Claude Code and compatible agents.

## Installation

```bash
npx skills add antstackio/skills
```

## Contributing

See [AGENTS.md](./AGENTS.md) for guidelines on creating and submitting skills.
